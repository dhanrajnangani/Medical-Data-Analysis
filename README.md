# Medical-Data-history-project-
This project focuses on analyzing medical and patient data to uncover health trends, hospital admissions patterns, and BMI-based insights using Power BI.
📊 Medical Data Analysis Dashboard – Power BI

I designed this interactive Medical Data History dashboard to analyze patient health and hospital admissions using key metrics and visuals.

🔹 Key Insights Covered:

Total patients, admissions, doctors, average age, and BMI score

Patient distribution across BMI categories (Underweight, Normal, Overweight, Obese)

Gender-wise BMI comparison

Monthly admissions trend to identify patterns over time

Obesity status analysis with percentage split

🔹 Features:

Interactive filters for diagnosis, quarter, gender, and admission date

Clear KPIs for quick decision-making

Clean and professional layout for healthcare insights
